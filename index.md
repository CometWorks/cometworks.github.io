# [Cometworks](https://github.com/CometWorks)

### Solutions

- [Quasar](https://github.com/CometWorks/quasar) - Supervisor control plane with a Web UI to run Magnetar servers (Space Engineers Dedicated Servers with plugin support)
- [Pulsar](https://github.com/SpaceGT/Pulsar) - Plugin loader for the Space Engineers game (client) 
- [Magnetar](https://github.com/CometWorks/magnetar) - Server loader with CLI plugin support for hosting without a UI and for building custom orchestrations

### OS and runtime support
- Linux: .NET 10
- Windows: .NET 10 and .NET Framework 4.8
